Задача: создать обработчик POST запроса, который принимает на вход имя валюту и возвращает текущий курс к рублю(RUB).
Реализовать тест, который проверяет корректность работы.

Описать в README.md:
1. Как запускать код.
2. Как отправлять запрос и формат ответа. 

Код запускается с помощью команды python3 main.py. Тесты с командой test.py.
Для отправки запроса нужно послать json файл в формате "currency" : "<валюта>".
Ответ получим в виде json файла: 

{
	"currency": "BYN",
	"rate": 28.4504
}
